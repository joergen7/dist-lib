# dist-lib

A library for distance-based algorithms.

## Installation

    make install

## Usage

```racket
#lang racket/base
(require dist-lib)
```

## Features

### Hierarchical Clustering

- min
- max
- WPGMA
- UPGMA

### Dynamic Programming

- Levenshtein distance
- Longest Common Subsequence (LCS)
